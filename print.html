  • Experience. 项目经验

    • TodoList - 单页在线todo应用

      TodoList 是一个轻量的在线单页待办事项管理应用，支持注册、登录、使用注册邮箱找回密码，可在线对待办事项进行分组、增删改查等管理操作，对分组也同样支持增删改查等管理操作，云同步。

      涉及技术栈：React + ES6 + Webpack + CSS

    • 移动端音乐播放器

      这个项目是一个单页面的移动端音乐播放器，布局和UI使用了QQ音乐的移动版界面， 包含的功能有：首页的推荐，排行榜以及音乐搜索功能。播放器界面包含 歌手头像、歌曲信息、歌词展示、歌曲进度条展示、循环播放等功能。播放器的数据 由后台模拟腾讯的请求获取，可以实时更新页面的展示和搜索的数据。

      涉及技术：ES6 + Sass + Fetch + Express + CORS + Webpack + Babel + Autoprefixer

    • Search Photos Wall

      一个具备搜索功能的照片墙，在顶部搜索框输入关键词，键入Enter键会得到一屏幕 由木桶布局组成的照片墙。滚动滚轮底部曝光会加载下一页。图片数据API 由pixabay提供， 图片墙具备懒加载、节流、回到顶部功能。

      涉及技术：ES6 + Ajax + Lazyload + throttle + CSS

    • 中国历史时间轴（公元后）

      这个项目是一个介绍中国历代王朝的网页，点击向前、向后按钮或者键盘的左右键可以翻页， 点击王朝名称可以看到王朝的简介。项目使用了CSS3动画和媒体查询， 不同王朝的间距长短反应了时间间隔的远近。资料参考百度百科中国历代王朝介绍。

      涉及技术栈：jQuery + CSS3 + Autoprefixer
